## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/yourusername/StayFinder.git

# Install backend dependencies
cd api
npm install

# Install frontend dependencies
cd client
npm install

# Start backend server
cd api
npm run dev

# Start frontend development
cd client
npm run dev
```
